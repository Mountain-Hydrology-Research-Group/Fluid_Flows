Resources to Learn Python Programming

This page provides links, videos, articles, and tutorials to help you start with Python programming.

Installing and Getting Started:

This section has links to download programs for this class and tutorials, articles, and videos on how to install and/or get started with these programs.

I recommend reading/watching the articles/videos on installing programs before downloading the programs if you are unfamiliar to get a better understanding of how these programs are set up. This may save you from needing to uninstall/reinstall programs later.

Unix Shell The Unix Shell allows you to interface with your computer using the command prompt. You can use the shell to run and download programs. See the tutorial for more on how and why to use the Unix Shell.
Tutorial:Unix Shell
Python and Anaconda Anaconda is a distribution for Python. It provides the Python language, packages, and package management. Overall, Anaconda makes Python easier to use. When you install Anaconda, you will also install Python. If you are new to Anaconda and/or Python, I recommend the video and/or article to the right which explain how to install and set up Anaconda as well as how Anaconda works. When installing Anaconda/Python, you should install the latest version (at least Python 3).
Link:Install Anaconda
Video:How to install Anaconda and Python
Article:How to install Anaconda and Python
Jupyter Notebooks Jupyter Notebooks are a way to write and run Python code. After installing Anaconda, you will have access to Jupyter Notebooks. They allow you to break up code into "cells" and run these cells individually. See the article on the right for more information on getting started with Jupyter Notebooks, including how to access them.
Article:How to use Jupyter Notebooks
Sublime Text Sublime Text is a text editor. It is a way to view and edit your code. There are many text editors that you may use, but Sublime Text is a good option.
Link:Download Sublime Text
Article:How to download/use Sublime Text
