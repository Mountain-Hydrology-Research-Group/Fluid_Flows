Resources to Learn Python Programming

Here I provide some links to download and install the programs necessary for this class. I also provide some tutorials that walk you through how to do so. I recommend reading/watching these tutorials before downloading the programs to get a better understanding of how these programs are set up. This may save you from needing to uninstall/reinstall programs later.

Setting Up and Using:

Python and Anaconda Anaconda is a distribution for Python. It provides the Python language, packages, and package management. Overall, Anaconda makes Python easier to use. When you install Anaconda, you will also install Python. If you are new to Anaconda and/or Python, I recommend the video and/or article to the right which explain how to install and set up Anaconda as well as how Anaconda works. When installing Anaconda/Python, you should install the latest version (at least Python 3). • Link to install Anaconda
• Video on how to install Anaconda and Python
• Documentation on how to install Anaconda and Python
Jupyter Notebooks Jupyter Notebooks are a way to write and run Python code. After installing Anaconda, you will have access to Jupyter Notebooks. They allow you to break up code into "cells" and run these cells individually. See the article on the right for more information on getting started with Jupyter Notebooks, including how to access them.
  • Article on how to use Jupyter Notebooks
Sublime Text Sublime Text is a text editor. It is a way to view and edit your code. There are many text editors that you may use, but Sublime Text is a good option.
