Lab 1.0: Setting Up Python

1. Setting up Anaconda

Anaconda is a Python distribution. It provides the Python language, packages, and package management. Overall, Anaconda makes Python easier to use. When you install Anaconda, you will also install Python.

To install Anaconda, click the link from the list below for your operating system (Windows, Mac, or Linux). This will take you to the documentation for installing Anaconda. Follow the directions carefully and download the latest version of Python (3.7). The top link will take you to this page: https://www.anaconda.com/distribution/#linux. Be sure to select the correct operating system in the middle of the page before downloading.

If you would like, you can also follow along with this video.
Note: If you are on Windows, use Anaconda Prompt instead of Windows Command prompt as he does in the video.
