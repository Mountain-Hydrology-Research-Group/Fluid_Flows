Lab 1.0: Setting Up Python

1. Setting Up Anaconda

Anaconda is a Python distribution. It provides the Python language, packages, and package management. Overall, Anaconda makes Python easier to use. When you install Anaconda, you will also install Python.

  1. Downloading Anaconda
    • Open https://www.anaconda.com/distribution.
    • Scroll down to this part of the screen (below). In the middle of the page, choose your operating system (Windows, Mac, Linux). Make sure that the correct operating system is selected before downloading Anaconda (boxed in blue).
    • Click the "Download" button for the Python 3.7 version (boxed in red).
  2. Installing Anaconda
    • Click the link from the list below for your operating system (Windows, Mac, or Linux). This will take you to the documentation for installing Anaconda. You have already completed Step 1 in this documentation by downloading Anaconda. Follow the rest of the directions carefully.
